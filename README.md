# Python Practice Programs 🐍

This repository contains a collection of Python practice programs organized into different categories. It is designed for beginners to learn Python programming, strengthen problem-solving skills, and practice pattern-based questions.

## 📂 Repository Structure

```text
Python-Practice/
│
├── Basic Programs/
│   └── Fundamental Python programs and beginner-level examples.
│
├── Normal Patterns/
│   └── Star and character pattern programs.
│
├── Number Patterns/
│   └── Number-based pattern programs.
│
└── README.md
```

## 📁 Folder Overview

### 📂 Basic Programs
This folder includes programs covering the basics of Python, such as:
- Variables and Data Types
- Input and Output
- Operators
- Conditional Statements (`if`, `elif`, `else`)
- Loops (`for`, `while`)
- Functions
- Lists, Tuples, Dictionaries, and Sets
- Strings
- Basic Problem Solving

### 📂 Normal Patterns
This folder contains pattern-printing programs using:
- Stars (`*`)
- Alphabets
- Triangles
- Pyramids
- Diamonds
- Other character-based patterns

### 📂 Number Patterns
This folder contains number pattern programs, including:
- Increasing Number Patterns
- Decreasing Number Patterns
- Floyd's Triangle
- Number Pyramids
- Pascal's Triangle
- Zig-Zag Number Patterns
